# Christmas Under The Stars

A beautiful website for a Church Christmas event, inspired by the event poster. This website is designed to be deployed on GitHub Pages.

## Features

- **Home Page**: Overview of the event with key details
- **Event Schedule**: Detailed timeline of activities and performances
- **Interactive Map**: Navigate the event venue with an interactive map
- **Photo Gallery**: Browse photos from previous events
- **Q&A Section**: Submit questions about the event

## Technologies Used

- HTML5
- CSS3 (with animations and parallax effects)
- JavaScript (vanilla)
- SVG for interactive map
- Responsive design for all device sizes

## Design Elements

- Deep blue starry background inspired by the event poster
- Hanging star decorations
- Glowing text effects
- Parallax scrolling
- Interactive elements

## Pages

1. **Home** (`index.html`): Main landing page with event overview
2. **Schedule** (`schedule.html`): Timeline of activities and performances
3. **Map** (`map.html`): Interactive venue map and location information
4. **Photos** (`photos.html`): Photo gallery with filtering and lightbox

## Setup and Deployment

This website is designed to be easily deployed to GitHub Pages:

1. Fork this repository
2. Go to repository settings
3. Navigate to Pages section
4. Select main branch as source
5. Save to deploy

## Local Development

To run this website locally:

1. Clone the repository
2. Open `index.html` in your browser

No build process or dependencies required!
